# theoretical-neuroscience

Minimal API docs for the core modules. The project is structured to grow
chapter-by-chapter alongside *Theoretical Neuroscience* (Dayan & Abbott).

Highlights:
- `tneuro.spiketrain` — spike trains and basic statistics
- `tneuro.neurons` — neuron models (starting with LIF)
- `tneuro.information` — information theory helpers
- `tneuro.encoding` — encoding/GLM utilities
- `tneuro.utils.plot` — optional plotting helpers (requires matplotlib)
